Avaliação de expressões para boolean

  • Valores que resultam em False
  • None
  • False
  • Zero de qualquer tipo numérico: 0, 0.0, 0L
  • Qualquer sequência vazia: '', (), []
  • Dicionário vazio: {}
  • Todos os outros valores resultam em True

Operadores booleanos

x or y  # OU, equivalente ao || de outras linguagens
x and y # E, equivalente ao && de outras linguagens
not x   # Não, equivalente ao !

x < y   # Menor que
x <= y  # Menor ou igual
x > y   # Maior que
x >= y  # Maior ou igual
x == y  # Igual
x != y  # Diferente
x is y      # Testa se é o mesmo objeto
x is not y  # Testa se é um objeto diferente
